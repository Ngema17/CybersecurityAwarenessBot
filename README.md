# Cybersecurity Awareness Chatbot

## Project Overview
This project involves the development of a **Cybersecurity Awareness Chatbot** that educates users about cybersecurity topics such as **password safety**, **phishing**, **scams**, and **safe browsing**. The chatbot is built using **C#** and is intended to be a console application that interacts with users in a conversational manner. It features a **voice greeting**, **ASCII art**, **personalized responses**, and **input validation**.

## Features
- **Voice Greeting**: The chatbot greets the user with a personalized voice message at the start.
- **User Input**: The chatbot asks the user for their name and responds to queries about cybersecurity topics.
- **ASCII Art**: Displays a header and themed art to enhance user experience.
- **Input Validation**: Handles invalid or empty user inputs and prompts the user to enter valid data.
- **Cybersecurity Responses**: Provides helpful tips and explanations regarding cybersecurity practices.
